Solver Simplex y Método Gráfico

Ingresa la función objetivo y las restricciones. El sistema resuelve el ejercicio con tablas simplex paso a paso y muestra solución gráfica cuando sea posible.

Funciona en tu navegador sin internet

1. Configuración del modelo

Selecciona el tipo de problema y define cuántas variables y restricciones tendrá.

Puedes ingresar enteros, decimales o fracciones. Ejemplo: 3, 2.5, 1/5.

Signos automáticos: si eliges Maximizar, las restricciones se colocan como <=. Si eliges Minimizar, se colocan como >=.

2. Función objetivo y restricciones

Llena los coeficientes de cada variable y el recurso de cada restricción.
